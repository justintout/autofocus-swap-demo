autofocus Swap Demo

In the demo to the left, you can add input fields to the list.

When you click on the "Add hyperscript input" button, it triggers an htmx swap to get a hyperscript-based input. The htmx swap respects the new input's autofocus attribute as expected. When you press Enter key inside a hyperscript-based input, it will use hyperscript to add another. This swap will not respect the new element's autofocus attribute.

When you click on the "Add htmx input" button, it triggers an htmx swap to get an htmx-based input. The htmx swap respects the new input's autofocus attribute as expected. When you press Enter key inside an htmx-based input, it will use hx-trigger to add another. This swap will respect the new element's autofocus attribute.

The difference here is that hyperscript fetch/put is much more raw than htmx swaps.
